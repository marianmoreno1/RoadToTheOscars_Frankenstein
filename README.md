# Frankenstein (2025) — Web Scraping & Audience Insights for Oscar Campaign

Repositorio con el código y los outputs del análisis de reseñas y conversación online sobre *Frankenstein (2025)*, con el objetivo de extraer **insights accionables** para diseñar una estrategia de campaña orientada a los Oscars.

## Qué incluye

- Web scraping de reseñas y contenido público desde varias fuentes
- Limpieza y normalización de datasets
- Diccionario de palabras para la clasificación por categoría de Oscars
- Análisis exploratorio (frecuencias, categorías, Top Critics vs Users)
- Análisis de sentimiento y extracción de temas (palabras, bigramas, conceptos)
- Generación de gráficos y tablas usadas en la presentación final

## Fuentes analizadas

- **Rotten Tomatoes** (Top Critics + User Reviews): principal fuente de insights por su valor crítico y estructura de reseñas  
- IMDb: apoyo para percepción general de audiencia
- YouTube: comentarios para señales cualitativas (tono/conversación)
- Google News: cobertura mediática y framing editorial

> Nota: el análisis prioriza las conclusiones que emergen de Rotten Tomatoes y se contrastan con el resto.

## Outputs principales

- Distribución de reseñas por categoría (Screenplay/Story, Acting, Cinematography/Visuals, Production Design/Costume, Music/Sound)
- Sentimiento promedio por categoría y por fuente
- Comparativa Top Critics vs Users (polaridad y subjetividad)
- Top palabras, bigramas y conceptos temáticos
- Ranking de nombres propios más mencionados
